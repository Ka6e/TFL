
Проект создан с целью ознакомления c процессом создания интерпретатора с виртуальной машиной на C#

## Клонирование проекта

Клонирование без авторизации в Github:

```bash
git clone https://github.com/Ka6e/TFL.git
```

Если у вас есть аккаунт Github и вы настроили SSH-ключи, то можно клонировать следующим образом:

```bash
git clone git@github.com:Ka6e/TFL.git
```

## Сборка

Сборка консольной утилитой dotnet из .NET SDK:

```bash
# Сборка
dotnet build

# Запуск тестов
dotnet test
```

## Структура проекта

- /docs - спецификация языка программирования Swiston и примеры программ 
- /src - исходный код на C# интепретатора
- /tests - тесты к интерпретатору, модульные и приёмочные
- .editorconfig - конфигурационный файл для форматирования кода
- Directory.Build.props - общие параметры сборки
- compiler.sln - файл решения
