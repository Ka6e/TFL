# Factorial

## Пример кода на языке Swiston:

>_Данная программа читает целое число и печатает его факториал, используя итеративный алгоритм_

```
func factorial(n: int): int {
    var result: int = 1;
    while (n > 1) {
        result = result * n;
        n = n - 1;
    }
    return result;
}

main {
    var n: int;
    read(n);
    print(factorial(n)); # для n=5 вывод: 120
}
```

**Демонстрируемый функционал:**
- Объявление пользовательской функции (`func`) с параметром и возвращаемым значением
- Цикл `while` с оператором сравнения `>`
- Переменные внутри функции и изменение параметра (параметры передаются по значению и изменяемы)
- Вызов пользовательской функции из `main`
