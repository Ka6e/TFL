# GCD

## Пример кода на языке Swiston:

>_Данная программа читает два целых числа и печатает их наибольший общий делитель (НОД), используя алгоритм Евклида_

```
func gcd(a: int, b: int): int {
    while (b != 0) {
        var tmp: int = b;
        b = a % b;
        a = tmp;
    }
    return a;
}

main {
    var a: int;
    var b: int;
    read(a);
    read(b);
    print(gcd(a, b)); # для a=12, b=8 вывод: 4
}
```

**Демонстрируемый функционал:**
- Объявление пользовательской функции (`func`) с несколькими параметрами и возвращаемым значением
- Цикл `while` с оператором `!=`
- Оператор остатка от деления `%`
- Вызов пользовательской функции из `main`
