# 01 — Лексическая структура языка Swiston --

# 1. Общие положения
	*	Исходный файл — текст в кодировке UTF-8.
	*	Программа состоит из одного файла.
	*	Лексический анализ выполняется до синтаксического.
	*	Регистр чувствителен.

## Лексемы делятся на:
	*	идентификаторы
	*	ключевые слова
	*	литералы
	*	операторы
	*	разделители
	*	комментарии
	*	пробельные символы

⸻

# 2. Пробельные символы

Пробельные символы не имеют семантического значения и служат для разделения лексем.

Допустимые пробельные символы:

U+0020  (space)
U+0009  (tab)
U+000A  (LF)
U+000D  (CR)


⸻

# 3. Комментарии

Поддерживаются однострочные комментарии.

Синтаксис:

" # комментарий до конца строки "

Комментарий начинается с # и продолжается до перевода строки.

⸻

# 4. Идентификаторы

Правила
	*	Начинаются с буквы латинского алфавита или _
	*	Далее могут содержать буквы, цифры и _
	*	Не могут совпадать с ключевыми словами

Примеры:

x
_value
counter1

# 6. Литералы

## 6.1 Целочисленные литералы (int)
	*	Десятичные
	*	Допускается знак -

Примеры:

0
42
-15

⸻

## 6.2 Литералы с плавающей точкой (float)
	*	Обязательная дробная часть
	*	Обязательная точка

Примеры:

3.14
-0.5
10.0


⸻

## 6.3 Строковые литералы (string)

- Оформление: двойные кавычки `"..."`.
- Поддерживаются экранированные последовательности: `\n, \t и.т.п.`
- Хранение: `UTF-8`.
- Строки поддерживают подмножество Unicode

Поддерживаются escape-последовательности:

Последовательность	Значение
\"	кавычка
\\	обратный слэш
\n	новая строка
\t	табуляция

Пример:

"Hello\nWorld"

Примеры в коде:

```
let a = "Hello";
let b = "Привет, мир\n";
let c = "Helloy: \t word";
```

## 9. Полная ISO EBNF-грамматика лексем

letter          = "A"…"Z" | "a"…"z" ;
digit           = "0"…"9" ;

identifier      = ( letter | "_" ), { letter | digit | "_" } ;

int_literal     = [ "-" ], digit, { digit } ;

float_literal   =
    [ "-" ],
    digit, { digit },
    ".",
    digit, { digit } ;

escape_sequence =
      "\\\""
    | "\\\\"
    | "\\n"
    | "\\t" ;

string_char     =
      ? any UTF-8 character except " and \ ?
    | escape_sequence ;

string_literal  =
    "\"", { string_char }, "\"" ;

bool_literal    = "true" | "false" ;

comment         =
    "#",
    { ? any character except newline ? } ;

whitespace      =
      " "
    | "\t"
    | "\n"
    | "\r" ;

plus            = "+" ;
minus           = "-" ;
star            = "*" ;
slash           = "/" ;
percent         = "%" ;

and_op          = "&&" ;
or_op           = "||" ;
not_op          = "!" ;

eq_op           = "==" ;
neq_op          = "!=" ;
lt_op           = "<" ;
le_op           = "<=" ;
gt_op           = ">" ;
ge_op           = ">=" ;

assign_op       = "=" ;

lparen          = "(" ;
rparen          = ")" ;
lbrace          = "{" ;
rbrace          = "}" ;
comma           = "," ;
semicolon       = ";" ;
colon           = ":" ;

delimiter =
      lparen
    | rparen
    | lbrace
    | rbrace
    | comma
    | semicolon
    | colon ;
