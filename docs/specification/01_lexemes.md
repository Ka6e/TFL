# Лексическая структура языка Swiston --

## 1. Общие положения

-   Исходный файл --- текст в кодировке UTF-8.
-   Программа состоит ровно из одного файла.
-   Язык регистрозависимый.
-   Лексический анализ выполняется до синтаксического анализа.
-   Комментарии и пробельные символы не участвуют в синтаксическом
    анализе.

------------------------------------------------------------------------

## 2. Классификация лексем

  Класс                Назначение
  -------------------- -------------------------------
  Идентификаторы       Имена переменных
  Ключевые слова       Зарезервированные слова языка
  Литералы             Константные значения
  Операторы            Арифметические и строковые
  Разделители          Структурные символы
  Комментарии          Игнорируются компилятором
  Пробельные символы   Разделяют лексемы

------------------------------------------------------------------------

## 3. Пробельные символы

Пробельные символы не имеют семантического значения и служат для разделения лексем.

Допустимые пробельные символы:

  Символ            Unicode
  ----------------- ---------
  Пробел            U+0020
  Табуляция         U+0009
  Перевод строки    U+000A

### ISO EBNF

``` ebnf
whitespace = " " | "\t" | "\n"
```

------------------------------------------------------------------------

## 4. Комментарии

Поддерживаются однострочные комментарии.

Синтаксис:

" # комментарий до конца строки "

Комментарий начинается с `#` и продолжается до перевода строки.

⸻

## 5. Идентификаторы

### Правила
  - Начинаются с буквы латинского алфавита или `_`
  - Далее могут содержать буквы, цифры и `_`
  - Не могут совпадать с ключевыми словами

### EBNF

``` ebnf
letter      = "A"…"Z" | "a"…"z" ;
digit       = "0"…"9" ;

identifier  =
    ( letter | "_" ),
    { letter | digit | "_" } ;
```

Примеры:

x
_value
counter1


------------------------------------------------------------------------

## 6. Ключевые слова

Зарезервированные слова:

`main`
`var`
`const`
`read`
`print`
`int`
`float`
`string`


Ключевые слова не могут использоваться как идентификаторы.

## 7. Литералы

### 7.1 Целочисленные литералы (int)
   - Десятичные
   - Допускается знак `-`

``` ebnf
int_literal =
    [ "-" ],
    digit,
    { digit } ;
```

Примеры:

0
42
-15

------------------------------------------------------------------------

### 7.2 Литералы с плавающей точкой (float)
   - Обязательная дробная часть
   - Обязательная точка

``` ebnf
float_literal =
    [ "-" ],
    digit, { digit },
    ".",
    digit, { digit } ;
```

Примеры:

3.14
-0.5
10.0

------------------------------------------------------------------------

### 7.3 Строковые литералы (string)

   - Оформление: двойные кавычки `"..."`.
   - Поддерживаются экранированные последовательности: `\n, \t и.т.п.`
   - Хранение: `UTF-8`.
   - Строки не поддерживают подмножество Unicode

Поддерживаются escape-последовательности:

#### Escape-последовательности

  Последовательность   Значение
  -------------------- ---------------
  `"`                 Кавычка
  `\`                 Обратный слеш
  `\n`                 Новая строка
  `\t`                 Табуляция


``` ebnf
backslash = "\" ;
quote     = '"' ;

escape_sequence =
      backslash, quote
    | backslash, backslash
    | backslash, "n"
    | backslash, "t" ;

string_character =
       любой символ UTF-8 кроме " и \
    | escape_sequence ;

string_literal =
    quote,
    { string_character },
    quote ;
```

Пример:

"Hello\nWorld"

------------------------------------------------------------------------

## 8. Операторы

Категория        Операторы
  ---------------- -------------------
  Арифметические   `+` `-` `*` `/` `%`
  Строковые        `+` (конкатенация)
  Сравнение        `==` `!=`
  Присваивание     `=`

``` ebnf
plus = "+" ;
minus = "-" ;
star  = "*" ;
slash = "/" ;
percent = "%" ;

eq_op = "==" ;
neq_op = "!=" ;
assign_op = "=" ;
```

------------------------------------------------------------------------

## 9. Разделители

  Символ     Назначение
  -------- ---------------------------------------
  `{` `}`   Границы блока кода 
  `(` `)`   Скобки для выражений и вызовов функций 
  `,`       Разделитель элементов 
  `;`       Завершение инструкции 
  `:`       Разделитель типа в объявлениях 

``` ebnf
lparen = "(" ;
rparen = ")" ;
lbrace = "{" ;
rbrace = "}" ;
comma  = "," ;
semicolon = ";" ;
colon  = ":" ;
```

------------------------------------------------------------------------


## 9. Полная EBNF-грамматика лексем

``` ebnf
letter = "A"…"Z" | "a"…"z" ;
digit  = "0"…"9" ;

identifier =
    ( letter | "_" ),
    { letter | digit | "_" } ;

int_literal =
    [ "-" ],
    digit,
    { digit } ;

float_literal =
    [ "-" ],
    digit, { digit },
    ".",
    digit, { digit } ;

backslash       = "\" ;
quote           = '"' ;

escape_sequence =
      backslash, quote
    | backslash, backslash
    | backslash, "n"
    | backslash, "t" ;

string_character = любой символ Unicode кроме quote и backslash
    | escape_sequence ;

string_literal =
    quote,
    { string_character },
    quote ;

comment =
    "#",
    { любой символ кроме "\n"  } ;

whitespace =
      " "
    | "\t"
    | "\n" ;

plus            = "+" ;
minus           = "-" ;
star            = "*" ;
slash           = "/" ;
percent         = "%" ;

eq_op           = "==" ;
neq_op          = "!=" ;

assign_op       = "=" ;

lparen          = "(" ;
rparen          = ")" ;
lbrace          = "{" ;
rbrace          = "}" ;
comma           = "," ;
semicolon       = ";" ;
colon           = ":" ;

delimiter =
      lparen
    | rparen
    | lbrace
    | rbrace
    | comma
    | semicolon
    | colon ;
```
