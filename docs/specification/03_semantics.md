# Семантика языка Swiston

## 1. Общие свойства

Swiston — императивный, статически типизированный язык.

| Свойство                     | Описание                   |
| ---------------------------- | -------------------------- |
| Неявные преобразования типов | Запрещены                  |
| Передача параметров          | Не применимо (функций нет) |

------------------------------------------------------------------------

## 2. Базовые типы данных

### 2.1 int
   - 32-битное знаковое целое
   - Поддерживает арифметику и сравнение

------------------------------------------------------------------------


## 2.2 float
   - 64-битное число с плавающей точкой
   - Поддерживает арифметику и сравнение
   - Неявного преобразования int → float нет

------------------------------------------------------------------------

## 2.3 string
   - Последовательность UTF-8 символов
   - Поддерживает конкатенацию `+` и сравнение `==` и `!=`

Встроенные функции:

length(s: string): int
substr(s: string, start: int, len: int): string

------------------------------------------------------------------------

## Операторы сравнения

| Типы операндов    | Семантика                         |
| ----------------- | --------------------------------- |
| `int` с `int`     | Сравнение по численному значению  |
| `float` с `float` | Сравнение по значению IEEE 754    |
| Разные типы       | Запрещено (ошибка проверки типов) |


------------------------------------------------------------------------

## 3. Переменные

| Категория | Свойство     |
| --------- | ------------ |
| `var`     | Изменяемые   |
| `const`   | Неизменяемые |


Правила:

 1.  Переменная должна быть объявлена до использования.
 2.  Тип выражения при присваивании должен совпадать с типом переменной.
 3.  Повторное присваивание `const` запрещено.

------------------------------------------------------------------------

## 4. Области видимости
 1.  Каждый блок {} создаёт новую область
 2. Вложенные объявления скрывают внешние
 3.  Дублирование в одной области запрещено

------------------------------------------------------------------------


## 5. Присваивание
 1.  Тип выражения должен совпадать с типом переменной
 2.  Присваивание const запрещено

------------------------------------------------------------------------

## 6. Контекстно-зависимые правила
 1.	В программе должна существовать ровно одна функция `main`.
 2.	Нельзя использовать необъявленные идентификаторы.
 3.	Нельзя объявлять две сущности с одинаковым именем в одной области.
 4.	Тип возвращаемого значения должен совпадать с объявленным.
