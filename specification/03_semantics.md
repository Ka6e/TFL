03 — Семантика языка Swiston

1. Общие свойства

Swiston — императивный, статически типизированный язык.
	•	Все типы проверяются на этапе компиляции.
	•	Не существует неявных преобразований типов.
	•	Выполнение начинается с main.

⸻

2. Типы данных

2.1 int
	•	32-битное знаковое целое
	•	Поддерживает арифметику и сравнение

⸻

2.2 float
	•	64-битное число с плавающей точкой
	•	Поддерживает арифметику и сравнение
	•	Неявного преобразования int → float нет

⸻

2.3 string
	•	Последовательность UTF-8 символов
	•	Поддерживает:
	•	конкатенацию +
	•	сравнение == и !=

Встроенные функции:

length(s: string): int
substr(s: string, start: int, len: int): string


⸻

2.4 bool

Значения:

true
false

Операции:

&&  ||  !

Short-circuit семантика
	•	A && B → B вычисляется только если A == true
	•	A || B → B вычисляется только если A == false

⸻

3. Переменные

var
	•	Изменяемые
	•	Должны быть объявлены до использования

const
	•	Неизменяемые
	•	Обязательная инициализация
	•	Повторное присваивание запрещено

⸻

4. Области видимости
	•	Каждый блок {} создаёт новую область
	•	Вложенные объявления скрывают внешние
	•	Дублирование в одной области запрещено

⸻

5. Присваивание
	•	Тип выражения должен совпадать с типом переменной
	•	Присваивание const запрещено

⸻

6. Функции
	•	Параметры передаются по значению
	•	Типы аргументов должны совпадать
	•	Обязателен return, если тип не void
	•	Все пути исполнения должны возвращать значение

⸻

7. Ветвления
	•	Условие должно иметь тип bool
	•	Выполняется ровно одна ветка

⸻

8. Цикл while
	•	Условие должно иметь тип bool
	•	Проверяется перед каждой итерацией

break

Прерывает ближайший цикл.

continue

Переходит к следующей итерации.

⸻

9. Контекстно-зависимые правила
	1.	В программе должна существовать ровно одна функция main.
	2.	break и continue допустимы только внутри while.
	3.	Нельзя использовать необъявленные идентификаторы.
	4.	Нельзя объявлять две сущности с одинаковым именем в одной области.
	5.	Нельзя вызывать функцию до её объявления.
	6.	Все функции должны возвращать значение по всем путям исполнения.
	7.	Тип возвращаемого значения должен совпадать с объявленным.
