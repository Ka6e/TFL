# Список тестов

## Идентификаторы и ключевые слова

- [ ] Разбор идентификаторов: `alice Bob C00L D_2 x`
- [ ] `type  arrtype = array of int` — type, array, of
- [ ] `let var x := 0 in x end` — let, var, in, end
- [ ] `while isdigit(buffer) do break` — while, do, break
- [ ] `if n = 0 then 1 else n * factorial(n-1)` — if, then, else
- [x] `for i:=10 to 20 do printi(i)` — for, to
- [x] `function foo() = ()` — function
- [x] `if a=nil then b` — nil

## Литералы

- [x] Разбор литералов целых чисел: `0 1234 56789 00173`
- [x] Разбор строк: `"" "0" "Hello, world!"`
- [x] Разбор простых escape-последовательностей: `"\n\t\"\\"`
- [x] Разбор escape-последовательностей в каретной нотации: `"\^@" "\^A\^C\^Z" "\^[" "\^\" "\^]" "\^^" "\^_" "\^?"`
- [x] Разбор escape-последовательностей с номером ASCII-символа: `digits := "\048\049\050"`
- [x] Разбор многострочных литералов с пропуском пробельных символов между двумя `\`

## Операторы, разделители и скобки

- [x] `x + y / (10 - z * 2)` — операторы `+, -, *, /`, скобки `()`
- [x] `a < b | a > c | b = c` — операторы `<, >, =, |`
- [x] `a <= b & b >= c & a <> c` — операторы `<=, >=, <>, &`
- [x] `speed.x := v[0]` — операторы `.`, `:=`, скобки `[]`
- [x] `type list = {first: int, rest: list}` — скобки `{}`, разделители `':', ','`
- [x] `(foo(x);0)` — разделитель `;`

## Пробелы и комментарии

- [x] Пропуск пробельных символов: `' '`, `'\t'`, `'\r'`, `'\n'`, `'\f'`
- [x] Комментарии: `/* ... */`
- [x] Вложенные комментарии: `/* ... /* ... */ */`
