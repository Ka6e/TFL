# Список тестов лексера

## Идентификаторы

- [x] Простые идентификаторы: `x`, `foo`, `_bar`, `foo123`, `_`, `camelCase`
- [x] Несколько идентификаторов через пробел: `a b`
- [x] Идентификаторы с заглавными буквами: `alice Bob C00L D_2 x`

## Ключевые слова

- [x] Управляющие конструкции: `main`, `var`, `const`, `read`, `print`
- [x] Типы: `int`, `float`, `string`
- [x] Встроенные функции: `length`, `substr`
- [x] Несколько ключевых слов: `var const int`
- [x] Слово с суффиксом → идентификатор: `mainX`, `variable`, `consts`, `integer`, `reading`
- [x] Суффиксы length/substr → идентификатор: `lengths`, `length2`, `length_x`, `substrings`, `substr2`, `substr_x`
- [x] Суффиксы float/string → идентификатор: `floater`, `strings`

## Литералы

- [x] Целые числа: `0`, `1`, `42`, `1000`
- [x] Несколько целых чисел: `0 1234 56789 0017`
- [x] Вещественные числа: `0.0`, `1.5`, `3.14`
- [x] Несколько вещественных чисел: `0.0 1.5 3.14`
- [x] Строковые литералы: `""`, `"hello"`, `"hello world"`

## Операторы

- [x] Арифметические: `+`, `-`, `*`, `/`, `%`
- [x] Все арифметические вместе: `+ - * / %`
- [x] Сравнения: `==`, `!=`
- [x] Оба оператора сравнения: `== !=`
- [x] Присваивание: `=`
- [x] Отличие `=` от `==`: `= ==`
- [x] Унарный минус: `-42` → Minus + IntLiteral(42)

## Разделители

- [x] По одному: `,`, `;`, `:`, `(`, `)`, `{`, `}`
- [x] Скобки с пробелом: `( )`, `{ }`

## Комментарии

- [x] Одиночный комментарий `#` — пустой результат
- [x] Комментарий перед кодом
- [x] Комментарий в конце строки
- [x] Несколько комментариев подряд

## Пробельные символы

- [x] Пропуск `' '`, `'\t'`, `'\r'`, `'\n'`, `'\f'`

## Структура программы

- [x] Объявление `var x : int = 42;`
- [x] Объявление `const y : int = 0;`
- [x] Оператор `read(x);`
- [x] Оператор `print(a);`
- [x] Арифметическое выражение: `a + b * c - d / e % f`
- [x] Выражение со сравнением: `a == b != c`
- [x] Присваивание: `x = 10;`
- [x] Блок main с var и print
- [x] Программа с комментариями #
- [x] Объявление переменной с типом int: `const X: int = 102;`
- [x] Объявление переменной с типом float: `var x : float = 1.5;`
- [x] Объявление переменной с типом string: `var s : string = "hello";`
- [x] Вызов substr: `substr(s, 0, 3)`
- [x] Вызов length: `length(s)`
