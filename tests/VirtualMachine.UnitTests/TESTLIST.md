- [x] ReadI: одно значение

- [x] ReadI: два значения подряд
    - вход: 10, 20 → вывод "1020"
- [x] ReadI: значение попадает в ExitCode
    - вход: 777 → exitCode == 777
- [x] ReadF: одно значение
    - вход: 3.5 → вывод "3.5"
- [x] ReadF: два значения подряд
    - вход: 1.5, 2.5 → вывод "1.52.5"
- [x] ReadS: одно значение
    - вход: "hello" → вывод "hello"
- [x] ReadS: два значения подряд
    - вход: "foo", "bar" → вывод "foobar"

## Переменные

- [x] DefineVar + LoadVar: объявление и чтение переменной
    - x = 42 → вывод "42"
- [x] StoreVar: присваивание нового значения
    - x = 10, затем x = 20 → вывод "20"
- [x] Несколько переменных: a = 3, b = 4 → a + b = "7"
- [x] DefineVar строки: s = "hello" → вывод "hello"
- [x] DefineVar float: f = 3.14 → вывод "3.14"
- [x] PushVars/PopVars: вложенная область видимости
    - outer = 10, inner = 20 → "20", "10", "10"
- [x] PushVars/PopVars: присваивание во внешней области из вложенной
    - outer x = 1, вложенная область меняет x на 99 → вывод "99"

## Переходы

- [x] Jump: безусловный переход через инструкции
    - Push 42, Jump 3 (пропуск Push 99), Print → "42"
- [x] JumpIfTrue: переход при ненулевом условии
    - Push 1 (true), JumpIfTrue 4 (пропуск Push 0 и Jump), Push 42, Print → "42"
- [x] JumpIfTrue: нет перехода при нулевом условии
    - Push 0 (false), JumpIfTrue 4 (не срабатывает), Push 99, Print → "99"
- [x] JumpIfFalse: переход при нулевом условии
    - Push 0 (false), JumpIfFalse 4 (пропуск Push 0 и Jump), Push 42, Print → "42"
- [x] JumpIfFalse: нет перехода при ненулевом условии
    - Push 1 (true), JumpIfFalse 4 (не срабатывает), Push 99, Print → "99"
- [x] Реализация if/else через JumpIfFalse и Jump
    - if (1 < 5) { print 1 } else { print 0 } → "1"
